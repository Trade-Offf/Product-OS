# Examples

## Example 1: ASCII 弹窗线框

### Input

```text
把这段线框图转成高保真原型，不新增功能，只做视觉还原。

┌──────────────────────────────────────────────┐
│ 新增带单员 / 编辑带单员                 [X]   │
├──────────────────────────────────────────────┤
│ UID：            [____________________]      │
│ 带单员昵称：      [____________________]      │
│ 头像：            [上传图片] [预览]           │
│ 个人简介：        [____________________]      │
│                  [____________________]      │
│ 联系方式：        [____________________]      │
│ 状态：            ( ) 启用中  ( ) 已停用      │
│ 备注：            [____________________]      │
│                  [____________________]      │
├──────────────────────────────────────────────┤
│                          [取消] [保存]       │
└──────────────────────────────────────────────┘
```

### Expected Behavior

- 识别为后台弹窗表单
- 保留标题、关闭按钮、字段顺序、底部操作区
- 用更真实的输入框、上传区、单选按钮和按钮样式表现
- 不新增“角色权限”“校验提示”“分步骤”等未提及内容

## Example 2: 简单列表页草图

### Input

```text
根据下面线框做一个后台列表页原型：

+------------------------------------------------------+
| 订单管理                                   [新增订单] |
+------------------------------------------------------+
| 订单号 [______]  状态 [下拉]  用户UID [______] [查询] |
+------------------------------------------------------+
| 表格区域                                             |
| 列：订单号 / 用户 / 金额 / 状态 / 创建时间 / 操作      |
+------------------------------------------------------+
| 分页区域                                             |
+------------------------------------------------------+
```

### Expected Behavior

- 识别为后台列表页
- 顶部标题 + 操作按钮
- 中间搜索区
- 下方表格占位和分页
- 不额外补充筛选条件或批量操作

## Recommended Triggers

- `把这段线框图转成原型图`
- `根据这个线框做高保真原型`
- `按这个草图还原一个后台页面`
- `不要改结构，把这张线框做成正式原型`
